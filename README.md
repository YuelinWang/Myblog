# myblogvue

用于技术分享和总结的个人博客，兼容pc和移动端

使用 vue.js 框架和Element UI来实现前端组件及数据交互，使用 webpack 构建，另外使用shCore.js实现代码样式



## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
